# Technical Club Portal

A responsive **Technical Club Portal website** built using **HTML5, CSS3, and JavaScript** as part of the Web Technologies Lab Assignment.


## 📌 Project Overview

This project is a multi-page responsive website designed for a CSE Technical Club.  
It provides information about:

- Club Home Page
- Technical Events
- Team Members
- Contact / Join Us Form

The website focuses on **modern UI design, responsiveness, and JavaScript interactivity**.


## 🛠️ Technologies Used

- HTML5 (Semantic Structure)
- CSS3 (Flexbox, Grid, Animations, Responsive Design)
- JavaScript (DOM Manipulation, Form Validation, Theme Toggle)
- GitHub Pages (Deployment)


## 🎨 Features

✔ Responsive design for mobile, tablet, and desktop  
✔ Dark / Light theme toggle  
✔ Dynamic events generated using JavaScript  
✔ Team section with modern card UI  
✔ Contact form with client-side validation  
✔ Glassmorphism-inspired UI design  
✔ Smooth hover animations and transitions
